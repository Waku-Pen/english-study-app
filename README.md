# 英語学習AIパートナー 使い方

`英語学習AI.html` をダブルクリックしてブラウザで開くだけで使えます(インストール不要)。
**Chrome または Edge 推奨**(音声入力・読み上げ機能のため)。

## 最初にやること(1回だけ)

1. **⚙️ 設定タブ**を開く
2. 使うAIを選ぶ(Claude / Gemini どちらでも可、両方登録して切り替えもOK)
   - Claude APIキー: https://console.anthropic.com/ で取得(`sk-ant-...`)
   - Gemini APIキー: https://aistudio.google.com/apikey で取得(`AIza...`)
3. 自分の英語レベルを選ぶ
4. 「💾 保存」→「🔌 接続テスト」で ✅ が出れば準備完了

APIキーはこのPCのブラウザ内(localStorage)にのみ保存されます。

## 機能

### 🎤 スピーキング
- 「▶ セッション開始」でAIが英語で話しかけてきます
- 🎤ボタンで音声入力(英語で話すと文字になって送信されます)
- **日本語で入力すると「英語でどう言うか」を教えてくれます**
- 間違いは ✏️ でやさしく訂正、便利な表現は 💡 で紹介してくれます
- 「■ 終了して振り返りを保存」で、学んだ語彙と振り返りメモが自動保存されます

### 📓 日記・振り返り
- 今日あったことを日本語で書いて「✨ 英語化」ボタンを押すと:
  - 自然な英語版の日記
  - 使える語彙リスト(単語帳に自動登録)
  - 「今日学んだこと」振り返りメモ
  が作成・保存されます

### 📚 単語帳
- 日記・スピーキングで学んだ語彙が自動で貯まります
- 🔊 で発音を確認できます

### 🔁 シャドーイング
- 貯まった語彙を盛り込んだ練習用スクリプトをAIが生成
- **速度を 0.5x / 0.75x / 1.0x / 1.25x に変更可能**
- 全文連続再生、または文をクリックして個別再生
- 日本語訳の表示ON/OFF可

### 📊 週間レポート
- 直近7日間の記録から、成長ポイント・復習おすすめ・来週のアドバイスをまとめたレポートを作成

## データについて

- 全データ(日記・単語帳・レポート)はブラウザのlocalStorageに保存されます
- ⚙️設定タブの「⬇ エクスポート」でJSONファイルとしてバックアップできます
- ブラウザの履歴・サイトデータを削除すると学習データも消えるので、定期的なエクスポートをおすすめします
